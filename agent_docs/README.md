# agent_docs

This directory provides a standardized, AI-readable framework for documenting the `sqshandler` project.

## Structure
- [01_orientation/](file:///Users/jefflunt/code/sqshandler/agent_docs/01_orientation/README.md): High-level system context, onboarding, and architecture overview.
- [02_patterns/](file:///Users/jefflunt/code/sqshandler/agent_docs/02_patterns/README.md): Code style guidelines, high-precision logging conventions, and testing patterns.
- [03_deep_dives/](file:///Users/jefflunt/code/sqshandler/agent_docs/03_deep_dives/README.md): Detailed subsystem breakdowns and designs.
- [04_plans/](file:///Users/jefflunt/code/sqshandler/agent_docs/04_plans/): Past, current, and future feature plans and execution steps.
- [05_tech_support/](file:///Users/jefflunt/code/sqshandler/agent_docs/05_tech_support/README.md): Incident logs and root-cause analyses of tech support issues.
- [templates/](file:///Users/jefflunt/code/sqshandler/agent_docs/templates/): Templates for plans, patterns, deep dives, and tech support investigations.
